Pusat Bantuan
Automation
  1. Automation dapat di lakukan pada bagian home dan juga bagian router yang ada pada bagian menu kiri lebih dianjurkan melakukan automation di router
  2. Ada dua pilihan dalam automation yaitu Simple Bandwitd dan simple and akurat
  3. automation tidak bisa di lakukan bersamaan karena tetapi automation simple and akurat dapat diperbarui
  4. terdapat mengatasi msalah otomatis pada saat automation jadi tidak ada yang perlu di hawatirkan
Manual command
  1. pada manual command tidak memiliki hasil yang bagus untuk orang awam tapi tidak sesulit itu untuk dipahami
  2. terdapat form untuk router yang akan di remote pilih routernya,
  3. pada form yang kedua terdapat input untuk command yang akan di lakukan contoh command ada di bawah manual command bantuan ini
  4. setelah mengetik atau memilih command ada tombol untuk mengirim commnad. maka akan dilakukan proses secara paralel
Contoh Command untuk perihal bandwidth(manual commnad)
  1. "ip address print" => melihat daftar ip pada router
  2. "ip firewall magle print" => mengecek firewall mangle pada router
  3. "queue simple print" => pengecekan terhadap automasi simple bandwidth
  4. "queue type print" => pengecekan terhadap automasi (pendukung)
  5. "queue tree print" => pengecekan terhadap automasi simple dan akurat
  6. "system reboot" => untuk melakuan restart pada router jika terjadi permasalahan yang diperlukan restart
Router
  1. router ditambahkan direkomendasikan orang yang lenih tau tentang jaringan
  2. menambah router dilakukan dengan meng click icon tambah pada menu bagian kiri tepatnya di router
  3. mengisi propertinya di sesuakan dengan keadaan router
  4. mengeditan juga harap dilakukan yang lebih tau tentang jaringan
  5. penghapusan router dapat menghapus configurasi dan riwayatnya
